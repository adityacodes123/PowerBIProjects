# D-Mart Sales Analysis Dashboard (Power BI)

# Project Overview
This project analyzes D-Mart sales data to understand customer demographics, product performance, and overall sales trends. An interactive dashboard was developed using Power BI to transform raw data into meaningful insights that support business decision-making.

The dataset was cleaned and transformed using Power Query, and calculations were performed using DAX to generate key business metrics. The dashboard helps visualize important patterns in sales performance and customer behavior.

# Tools Used
- Power BI
- Power Query
- DAX
- Excel

# Dataset
The dataset contains information related to:
- Sales transactions
- Customer demographics
- Product categories
- Revenue and performance metrics

# Key Features
- Interactive dashboard for sales analysis
- Data cleaning and transformation using Power Query
- KPI cards to track important business metrics
- Visualizations including bar charts, donut charts, and gauge charts
- Product and customer performance analysis

#  Key Insights
- Identified top-performing product categories
- Analyzed customer demographics and buying patterns
- Monitored key performance indicators such as total sales and revenue
- Highlighted trends that help improve marketing and sales strategies
